# Dashboard Zelify

**Dashboard Zelify** es un dashboard administrativo construido con Next.js que cuenta con más de 200 componentes UI y plantillas pre-construidas, elementos, componentes, páginas, diseño de alta calidad, integraciones y mucho más para ayudarte a crear dashboards administrativos poderosos con facilidad.

## Descripción

**Dashboard Zelify** proporciona un conjunto diverso de componentes UI de dashboard, elementos, ejemplos y páginas necesarios para crear paneles administrativos o dashboards de primer nivel con características e integraciones **poderosas**. Ya sea que estés trabajando en una aplicación web compleja o un sitio web básico, **Dashboard Zelify** te tiene cubierto.

Al aprovechar las últimas características de **Next.js 15** y funcionalidades clave como **server-side rendering (SSR)**, **static site generation (SSG)**, e integración fluida de **rutas API**, **Dashboard Zelify** asegura un rendimiento óptimo. Con los beneficios adicionales de los avances de **React 18** y la confiabilidad de **TypeScript**, **Dashboard Zelify** es la elección definitiva para iniciar tu proyecto **Next.js** de manera eficiente.

## Instalación

```bash
npm install
```

Si estás usando **Yarn** como tu gestor de paquetes, el comando será:

```bash
yarn install
```

2. ¡Bien, ya casi estás ahí! Ahora todo lo que necesitas hacer es iniciar el servidor de desarrollo. Si estás usando **npm**, el comando es:

```bash
npm run dev
```

Y si estás usando **Yarn**, es:

```bash
yarn dev
```


## Tecnologías Utilizadas

- **Next.js 15** - Framework de React para producción
- **React 18** - Biblioteca de JavaScript para construir interfaces de usuario
- **TypeScript** - Superset tipado de JavaScript
- **Tailwind CSS** - Framework CSS utility-first
- **Prisma** - ORM moderno para bases de datos
- **NextAuth** - Autenticación para Next.js
- **ApexCharts** - Biblioteca de gráficos moderna
